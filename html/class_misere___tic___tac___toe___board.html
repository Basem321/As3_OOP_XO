Detailed Description

Represents a Misère Tic-Tac-Toe board with reversed win condition.

A standard 3×3 Tic-Tac-Toe board where getting three in a row means you LOSE rather than win. This creates a strategic game where players must balance offense and defense differently than traditional Tic-Tac-Toe.

Game rules:

  • 3×3 grid
  • Getting 3 in a row = you LOSE
  • Filling the board without 3 in a row = you WIN
  • No draws (if board fills, whoever didn't make 3-in-a-row wins)

Win/Lose logic:

  • is_lose(): Returns true if player made 3-in-a-row (they lose)
  • is_win(): Returns true if board full and player didn't make 3-in-a-row
