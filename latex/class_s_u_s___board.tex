\doxysubsection{Detailed Description}
Represents the SUS game board with scoring system. 

A 3×3 grid where\+:
\begin{DoxyItemize}
\item \doxylink{class_player}{Player} 1 places \textquotesingle{}S\textquotesingle{} letters
\item \doxylink{class_player}{Player} 2 places \textquotesingle{}U\textquotesingle{} letters
\item Points are scored by completing "{}\+SUS"{} patterns
\item Winner has most points when board is full
\end{DoxyItemize}

Scoring rules\+:
\begin{DoxyItemize}
\item When \textquotesingle{}S\textquotesingle{} is placed\+: Check for S-\/\+U-\/S patterns in all 8 directions
\item When \textquotesingle{}U\textquotesingle{} is placed\+: Check if it\textquotesingle{}s sandwiched between two \textquotesingle{}S\textquotesingle{}
\item Each "{}\+SUS"{} found = 1 point for the player who completed it
\end{DoxyItemize}

Patterns recognized\+:
\begin{DoxyItemize}
\item Horizontal\+: S-\/\+U-\/S (left-\/right)
\item Vertical\+: S-\/\+U-\/S (up-\/down)
\item Diagonal\+: S-\/\+U-\/S (both diagonals)
\end{DoxyItemize}

Game end\+: \doxylink{class_board}{Board} fills (9 moves total) Winner\+: \doxylink{class_player}{Player} with higher score Draw\+: Equal scores
