\doxysubsection{Detailed Description}
Defines classes for Misère Tic-\/\+Tac-\/\+Toe game variant. 

This file implements the Misère variant of Tic-\/\+Tac-\/\+Toe where the objective is reversed\+: the first player to get three in a row LOSES. Players must avoid creating three-\/in-\/a-\/row while trying to force their opponent into that position.
