\doxysubsection{Detailed Description}
Represents the Ultimate Tic-\/\+Tac-\/\+Toe meta-\/board. 

Manages\+:
\begin{DoxyItemize}
\item A 3×3 grid of mini-\/board states (main\+\_\+board)
\item Two \doxylink{class_mini_board}{Mini\+Board} instances for actual gameplay
\item Game flow logic (which board is active)
\item Win condition for the overall game
\end{DoxyItemize}

Game flow\+:
\begin{DoxyEnumerate}
\item \doxylink{class_player}{Player} selects which 3×3 position to play in
\item Plays a complete Tic-\/\+Tac-\/\+Toe game on that mini-\/board
\item Mini-\/board result is recorded on main\+\_\+board
\item Next player must play in position determined by previous move
\item First player to win 3 main-\/board positions in a row wins
\end{DoxyEnumerate}

Strategic depth\+:
\begin{DoxyItemize}
\item Your move position determines opponent\textquotesingle{}s board
\item Must balance winning current board vs forcing opponent to bad position
\item Closed boards force choice of new board
\end{DoxyItemize}
