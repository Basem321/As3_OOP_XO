\doxysubsection{Detailed Description}
Represents a Word Tic-\/\+Tac-\/\+Toe board with dictionary validation. 

A 3×3 grid where\+:
\begin{DoxyItemize}
\item Players place individual letters (A-\/Z)
\item Win by forming a valid 3-\/letter word
\item Words checked in all directions (rows, columns, diagonals)
\item Dictionary loaded from external file "{}dic.\+txt"{}
\end{DoxyItemize}

Game mechanics\+:
\begin{DoxyItemize}
\item Any letter can be placed in any empty cell
\item No restrictions on letter choice per player
\item First valid word wins immediately
\item Draw if board fills without forming valid words
\end{DoxyItemize}

Word validation\+:
\begin{DoxyItemize}
\item Must be exactly 3 letters
\item Must appear in loaded dictionary
\item Case-\/insensitive (converted to uppercase)
\item Checked after each move
\end{DoxyItemize}

Dictionary format\+:
\begin{DoxyItemize}
\item Plain text file with one word per line
\item Words converted to uppercase when loaded
\item Stored in set for O(log n) lookup
\end{DoxyItemize}

Checks if a valid word has been formed. 

After at least 3 moves, checks all 8 possible lines\+:
\begin{DoxyItemize}
\item 3 rows (horizontal)
\item 3 columns (vertical)
\item 2 diagonals
\end{DoxyItemize}

For each complete line (no empty cells), forms a 3-\/letter string and checks if it exists in the dictionary.
