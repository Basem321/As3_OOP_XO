\doxysubsection{Detailed Description}
Defines classes for Obstacles Tic-\/\+Tac-\/\+Toe game variant. 

This file implements a challenging Tic-\/\+Tac-\/\+Toe variant on a 6×6 board where random obstacles are added after every 2 moves, and players must connect 4 pieces in a row to win. The dynamic obstacles create an unpredictable and strategic gameplay experience.
