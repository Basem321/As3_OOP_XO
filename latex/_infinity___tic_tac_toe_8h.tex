\doxysubsection{Detailed Description}
Defines classes for Infinity Tic-\/\+Tac-\/\+Toe game variant. 

This file implements a unique Tic-\/\+Tac-\/\+Toe variant where each player can only have 3 pieces on the board at a time. After placing the 4th piece, the oldest piece is automatically removed, creating a dynamic game state.
