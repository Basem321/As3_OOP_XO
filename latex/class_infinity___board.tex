\doxysubsection{Detailed Description}
Represents an Infinity Tic-\/\+Tac-\/\+Toe board. 

A 3×3 board where each player can only maintain 3 pieces simultaneously. When a player places their 4th piece, their oldest piece is automatically removed, creating a constantly shifting game state. Players must achieve three-\/in-\/a-\/row before their pieces disappear.

Key features\+:
\begin{DoxyItemize}
\item Standard 3×3 grid
\item Maximum 3 pieces per player at any time
\item FIFO (First-\/\+In-\/\+First-\/\+Out) piece removal
\item Game continues indefinitely until someone wins
\item No draw condition (game always has a winner eventually)
\end{DoxyItemize}

Updates board with a player\textquotesingle{}s move. 

Places the piece at the specified position and adds it to the player\textquotesingle{}s move queue. If the player now has more than 3 pieces, automatically removes their oldest piece from the board.

Process\+:
\begin{DoxyEnumerate}
\item Validate position is empty and in bounds
\item Place new piece
\item Add position to player\textquotesingle{}s queue
\item If player has \texorpdfstring{$>$}{>}3 pieces, remove oldest from board and queue
\end{DoxyEnumerate}
