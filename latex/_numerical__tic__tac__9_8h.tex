\doxysubsection{Detailed Description}
Defines classes for Numerical Tic-\/\+Tac-\/\+Toe game variant. 

This file implements a strategic Tic-\/\+Tac-\/\+Toe variant where players use numbers instead of X and O. \doxylink{class_player}{Player} 1 uses odd numbers (1,3,5,7,9) and \doxylink{class_player}{Player} 2 uses even numbers (2,4,6,8). Players win by creating a line that sums to exactly 15.
