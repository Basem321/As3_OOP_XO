\doxysubsection{Detailed Description}
Represents a 6×6 board with dynamic obstacle placement. 

Players compete on a 6×6 grid where\+:
\begin{DoxyItemize}
\item Goal is to connect 4 pieces in a row (horizontal, vertical, or diagonal)
\item After every 2 moves (1 from each player), 1 random obstacle is added
\item Obstacles (\textquotesingle{}\#\textquotesingle{}) block cells permanently
\item Game is drawn if board fills with no winner
\end{DoxyItemize}

Obstacle mechanics\+:
\begin{DoxyItemize}
\item Placed randomly in empty cells
\item Cannot be removed or played on
\item Added automatically after move pairs
\item Adds strategic unpredictability
\end{DoxyItemize}

Win condition\+: 4 in a row in any direction Draw condition\+: \doxylink{class_board}{Board} full without any 4-\/in-\/a-\/row
