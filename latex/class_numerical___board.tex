\doxysubsection{Detailed Description}
Represents a Numerical Tic-\/\+Tac-\/\+Toe board. 

A 3×3 board where players place numbers instead of symbols. The game uses a magic square principle where any line (row, column, or diagonal) that sums to 15 wins the game.

Number allocation\+:
\begin{DoxyItemize}
\item \doxylink{class_player}{Player} 1 (symbol=1)\+: Uses odd numbers \{1, 3, 5, 7, 9\}
\item \doxylink{class_player}{Player} 2 (symbol=2)\+: Uses even numbers \{2, 4, 6, 8\}
\end{DoxyItemize}

Win condition\+:
\begin{DoxyItemize}
\item Any complete row, column, or diagonal that sums to exactly 15
\end{DoxyItemize}

Draw condition\+:
\begin{DoxyItemize}
\item All 9 cells filled without any sum of 15
\end{DoxyItemize}

Strategic depth\+:
\begin{DoxyItemize}
\item Each number can only be used once
\item Players must balance using small vs large numbers
\item Multiple potential winning lines to defend/attack
\end{DoxyItemize}
