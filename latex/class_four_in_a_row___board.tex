\doxysubsection{Detailed Description}
Represents a Connect Four game board. 

The board is a 6-\/row by 7-\/column grid where pieces fall to the lowest available position in the selected column. Players win by connecting four pieces in a row horizontally, vertically, or diagonally.

\doxylink{class_board}{Board} dimensions\+: 6 rows × 7 columns (42 total cells)

Game rules\+:
\begin{DoxyItemize}
\item Players alternate dropping pieces into columns
\item Pieces fall to the lowest available row in that column
\item First to connect 4 pieces in any direction wins
\item Game is a draw if board fills with no winner
\end{DoxyItemize}
