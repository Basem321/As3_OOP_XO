\doxysubsection{Detailed Description}
User interface for the SUS game. 

Handles player setup ensuring \doxylink{class_player}{Player} 1 gets \textquotesingle{}S\textquotesingle{} and \doxylink{class_player}{Player} 2 gets \textquotesingle{}U\textquotesingle{}, displays current scores during gameplay, and manages move input.
