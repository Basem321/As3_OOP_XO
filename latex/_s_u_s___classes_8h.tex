\doxysubsection{Detailed Description}
Defines classes for the SUS word-\/formation game. 

This file implements a unique 3×3 grid game where players place \textquotesingle{}S\textquotesingle{} and \textquotesingle{}U\textquotesingle{} letters trying to form the word "{}\+SUS"{} in any direction. Players score points for each "{}\+SUS"{} they complete, and the player with the most points when the board fills wins.
