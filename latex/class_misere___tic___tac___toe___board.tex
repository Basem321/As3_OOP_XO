\doxysubsection{Detailed Description}
Represents a Misère Tic-\/\+Tac-\/\+Toe board with reversed win condition. 

A standard 3×3 Tic-\/\+Tac-\/\+Toe board where getting three in a row means you LOSE rather than win. This creates a strategic game where players must balance offense and defense differently than traditional Tic-\/\+Tac-\/\+Toe.

Game rules\+:
\begin{DoxyItemize}
\item 3×3 grid
\item Getting 3 in a row = you LOSE
\item Filling the board without 3 in a row = you WIN
\item No draws (if board fills, whoever didn\textquotesingle{}t make 3-\/in-\/a-\/row wins)
\end{DoxyItemize}

Win/\+Lose logic\+:

\item \doxylink{class_misere___tic___tac___toe___board_aa875d0b9e0937d1194bdf3e7248f9480}{is\+\_\+win()}\+: Returns true if board full and player didn\textquotesingle{}t make 3-\/in-\/a-\/row
\end{DoxyItemize}
