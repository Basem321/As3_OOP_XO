\doxysubsection{Detailed Description}
Main entry point for the \doxylink{class_board}{Board} Game Collection. 

This file provides a menu-\/driven interface for selecting and playing various Tic-\/\+Tac-\/\+Toe variants and Connect Four. It handles game initialization, player setup, game execution, and cleanup for all supported game types.

Main function providing game selection menu. 

Displays a menu of available games, handles user selection, initializes the chosen game with appropriate board and \doxylink{class_u_i}{UI} classes, runs the game to completion, and performs cleanup.

Menu options (0-\/12)\+:
\begin{DoxyItemize}
\item 0\+: Exit program
\item 1\+: Four-\/in-\/a-\/\+Row (Connect Four)
\item 2\+: SUS Game
\item 3\+: 5×5 Tic-\/\+Tac-\/\+Toe
\item 4\+: Word Tic-\/\+Tac-\/\+Toe
\item 5\+: Misère Tic-\/\+Tac-\/\+Toe
\item 6\+: Diamond Tic-\/\+Tac-\/\+Toe
\item 7\+: 4x4 Tic-\/\+Tac-\/\+Toe
\item 8\+: Prymaid Tic-\/\+Tac-\/\+Toe
\item 9\+: Numerical Tic-\/\+Tac-\/\+Toe
\item 10\+: Obstacles Tic-\/\+Tac-\/\+Toe
\item 11\+: Infinity Tic-\/\+Tac-\/\+Toe
\item 12\+: Ultimate Tic-\/\+Tac-\/\+Toe
\end{DoxyItemize}

Game lifecycle for each selection\+:
\begin{DoxyEnumerate}
\item Create \doxylink{class_u_i}{UI} object
\item Create \doxylink{class_board}{Board} object
\item Setup players via \doxylink{class_u_i}{UI}
\item Create \doxylink{class_game_manager}{Game\+Manager}
\item Run game
\item Delete all dynamically allocated objects
\end{DoxyEnumerate}

Memory management\+:
\begin{DoxyItemize}
\item All dynamically allocated objects are properly deleted
\item \doxylink{class_player}{Player} array and individual players cleaned up after each game
\item \doxylink{class_board}{Board} and \doxylink{class_u_i}{UI} objects deleted before exit
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
0 on successful execution
\end{DoxyReturn}
\begin{DoxyNote}{Note}
Seeds random number generator at startup for computer players 

Invalid menu choices display error message and exit
\end{DoxyNote}
