\doxysubsection{Detailed Description}
Represents a 5×5 Tic-\/\+Tac-\/\+Toe board with counting victory. 

Unlike traditional Tic-\/\+Tac-\/\+Toe, this variant\+:
\begin{DoxyItemize}
\item Uses a 5×5 grid (25 cells total)
\item Game runs for exactly 24 moves (leaving 1 empty cell)
\item Players can create multiple three-\/in-\/a-\/row patterns
\item Winner is determined by counting total three-\/in-\/a-\/rows
\end{DoxyItemize}

Counting rules\+:
\begin{DoxyItemize}
\item Each unique sequence of 3 matching symbols counts as 1 point
\item Sequences checked in all directions\+: →, ↓, ↘, ↙
\item Overlapping patterns both count (e.\+g., XXXX has 2 patterns)
\end{DoxyItemize}

Example scoring\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{X\ X\ X\ .\ .\ \ \ ←\ This\ row\ has\ 1\ three-\/in-\/a-\/row}
\DoxyCodeLine{X\ X\ X\ X\ .\ \ \ ←\ This\ row\ has\ 2\ three-\/in-\/a-\/\mbox{\hyperlink{class_board_a948d978e3b5fc460559b588f5dee9572}{rows}}\ (positions\ 0-\/2\ and\ 1-\/3)}

\end{DoxyCode}


Game phases\+:
\begin{DoxyEnumerate}
\item Placement phase\+: Players alternate placing pieces (moves 1-\/24)
\item Scoring phase\+: Count all three-\/in-\/a-\/rows for each player
\item Victory\+: \doxylink{class_player}{Player} with more patterns wins
\end{DoxyEnumerate}
